This folder contains the art assets used across the various demos (to avoid cluttering with intermediate png files and so on).

## gb

Generate the `man.2bpp` file used in the `gb` demo using:

```
rgbgfx -o man.2bpp man-gb.png
```

Generate the `gbc.pal` file using:

```
rgbgfx -p gbc.pal gb-gbc-pal.png
```
